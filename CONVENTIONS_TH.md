# Thai Translation Style Guide (Modern CC0 Edition)

เอกสารนี้กำหนดแนวทางสำหรับการสร้างคำแปลพระไตรปิฎกฉบับภาษาไทยสมัยใหม่ (Modern Thai) เพื่อให้มีความสม่ำเสมอ แม่นยำ และทรงพลังในเชิงความหมาย

## 1. ระดับภาษา (Tone & Voice)
- **สไตล์:** กึ่งทางการ (Semi-formal) คล้ายบทความวิชาการหรือธรรมวิจัย
- **กลุ่มเป้าหมาย:** ผู้แสวงหาความรู้ทั่วไปและผู้ปฏิบัติธรรมที่ต้องการความชัดเจนในเชิงลึก
- **หลักการ:** ไม่เน้นราชาศัพท์ที่ยากเกินความจำเป็น แต่ต้องคงความเคารพ (Polite/Formal) และใช้ศัพท์เทคนิคทางธรรมอย่างแม่นยำ

## 2. แหล่งข้อมูลในการแปล (Source Priority)
1.  **Pali (Source of Truth):** ทุกประโยคต้องเทียบเคียงจากบาลีต้นฉบับ ไม่แปลต่อจากภาษาอังกฤษเด็ดขาด
2.  **English (Context):** ใช้คำแปลภาษาอังกฤษ (เช่น Sujato) เพื่อช่วยตีความบริบทและโครงสร้างประโยค
3.  **Dictionary:** อ้างอิงคำนิยามหลักจาก **พจนานุกรม พุทธศาสน์ ฉบับประมวลศัพท์ (ป.อ. ปยุตฺโต)**

## 3. ศัพท์ทางธรรมมาตรฐาน (Standard Terminology)

| Pali Term | Thai Translation | Note |
|-----------|-----------------|------|
| bhikkhave | ภิกษุทั้งหลาย | คำเรียกที่พระพุทธเจ้าใช้เรียกกลุ่มภิกษุ |
| Tathāgata | ตถาคต | ใช้เมื่อพระพุทธเจ้าตรัสถึงพระองค์เอง / หากผู้อื่นกล่าวถึง ให้ใช้ตามบริบท (เช่น "พระผู้มีพระภาค") |
| Dukkha | ทุกข์ | สภาพที่ทนได้ยาก / ความบีบคั้น |
| Samudaya | สมุทัย | เหตุเกิดแห่งความทุกข์ |
| Nirodha | นิโรธ | ความดับทุกข์ |
| Magga | มรรค / ปฏิปทา | ข้อปฏิบัติเพื่อให้ถึงความดับทุกข์ |
| Ariyasacca | อริยสัจ | ความจริงอันประเสริฐ |
| Taṇhā | ตัณหา | ความทะยานอยาก |
| Majjhimā Paṭipadā | มัชฌิมปฏิปทา | ทางสายกลาง / ข้อปฏิบัติสายกลาง |
| tiparivaṭṭaṁ dvādasākāraṁ | ไตรปริวัฏ 12 อาการ | ใช้ทั้งคำทับศัพท์และขยายความ (มีรอบ 3 มีอาการ 12) |

## 4. ข้อควรระวัง (Do's and Don'ts)
- **Do:** ใช้คำเชื่อมที่ช่วยให้ประโยคลื่นไหลแต่ไม่เยิ่นเย้อ
- **Do:** ใส่เชิงอรรถ (Footnotes) หากคำบาลีนั้นมีความหมายหลายนัย
- **Don't:** อย่าแปลแบบคำต่อคำ (Word-for-word) จนเสียอรรถรสภาษาไทย
- **Don't:** หลีกเลี่ยงการใช้ศัพท์สมัยใหม่ที่อาจส่อเค้าไปในทางบิดเบือนความหมายเดิม (เช่น การใช้ slang หรือศัพท์เทคนิคทางโลกที่ขัดกับธรรม)
